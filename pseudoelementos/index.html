::first-line (:first-line)

Selecciona la primera línea del elemento de contenido.

Los estilos sólo se aplicarán en la primera línea de este párrafo. Posteriormente, todo el texto tendrá el estilo normal.

La primera línea de este elemento no recibe ningún estilo especial, porque no es un elemento de bloque.

::grammar-error

Selecciona una parte del documento que contiene un error de gramática indicado por el navegador.

::selection

Selecciona la parte del documento que ha sido seleccionada.

::spelling-error

Selecciona una parte del documento que contiene un error de ortografía indicado por el navegador.
